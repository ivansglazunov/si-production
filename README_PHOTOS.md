```bash

sudo apt-get install imagemagick
```

### Обработка HEIC файлов:

После установки ImageMagick запустите:

```bash
python3 process_heic_only.py
```

Этот скрипт обработает все HEIC файлы из исходной папки `/home/ae/Загрузки/si_production_files` и добавит их в соответствующие пронумерованные папки.

## Формат и размер изображений

- **Формат:** WebP (оптимальный для веба - лучшее сжатие при хорошем качестве)
- **Размер:** 360x240px (соответствует размеру кадров в киноленте с учетом ретины-дисплеев)
- **Качество:** 85% (баланс между размером файла и качеством)
- **Метаданные:** Полностью удалены (включая геопозицию)
