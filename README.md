# SI Production

Next.js приложение с 3 полноэкранными секциями на одной странице.

## Установка

```bash
npm install
```

## Запуск

### Режим разработки

Для локальной разработки:

```bash
npm run dev
```

Или через tmux скрипт:

```bash
./start-dev.sh
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

### Production режим

Для production сборки:

```bash
npm run build
npm start
```

## Автозапуск через systemd

Для автоматического запуска сервера после перезагрузки системы используется systemd сервис.

### Установка сервиса

```bash
npm run service:install
```

Это установит сервис и включит автозапуск. Сервис будет запускать приложение в режиме разработки (`npm run dev`).

**Важно:** После завершения разработки необходимо переключить сервис на production режим:
1. Откройте файл `scripts/si-production.service`
2. Закомментируйте строку с `npm run dev` в `ExecStart`
3. Раскомментируйте строку с `npm start` в `ExecStart`
4. Также выполните `npm run build` перед перезапуском сервиса
5. Скопируйте обновленный файл: `sudo cp scripts/si-production.service /etc/systemd/system/`
6. Перезагрузите systemd: `sudo systemctl daemon-reload`
7. Перезапустите сервис: `npm run service:restart`

**Примечание:** Сервис запускает процесс напрямую через systemd. Для доступа к логам используйте:
```bash
# Просмотр логов в реальном времени
sudo journalctl -u si-production.service -f

# Последние 50 строк логов
sudo journalctl -u si-production.service -n 50
```

Если нужна tmux сессия для интерактивной работы во время разработки, используйте `./start-dev.sh`.

### Управление сервисом

```bash
# Запустить сервис
npm run service:start

# Остановить сервис
npm run service:stop

# Перезапустить сервис
npm run service:restart

# Проверить статус
npm run service:status

# Удалить сервис
npm run service:uninstall
```

### Доступ к приложению

После установки сервиса приложение будет доступно:
- Локально: [http://localhost:3000](http://localhost:3000)
- Через домен: [http://si-production.ru](http://si-production.ru) (требуется настроенный nginx)

Для подключения к tmux сессии:
```bash
tmux attach -t si-production
```

## Экраны

Все 3 экрана находятся на одной странице, переключение между ними происходит через вертикальный скролл:

1. **Первый экран** - Сетка 40x40 клеток с анимацией:
   - Линии сетки анимированно "прилетают" с краев экрана за 1 секунду
   - Цветные квадратики появляются через 1-5 секунд после начала анимации линий

2. **Второй экран** - Договор с кнопкой подписания и хлопушкой-нумератором

3. **Третий экран** - Галерея проектов и партнеров
