# SI Production

Next.js приложение с 3 полноэкранными секциями на одной странице.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Экраны

Все 3 экрана находятся на одной странице, переключение между ними происходит через вертикальный скролл:

1. **Первый экран** - Сетка 40x40 клеток с анимацией:
   - Линии сетки анимированно "прилетают" с краев экрана за 1 секунду
   - Цветные квадратики появляются через 1-5 секунд после начала анимации линий

2. **Второй экран** - Черный пустой экран

3. **Третий экран** - Черный пустой экран
